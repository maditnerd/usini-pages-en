Color sensor (TCS34725)

I have a sensor which detect color who was laying around, but it wasn't really great ( TCS3200)

So I bought a better version: The TCS34725.
This sensors use I2C.

 

Change led colors using a color sensor !
This is an @Arduino nano with TCS34725 color sensor and neopixels ring.

You can buy these on @Adafruit which is always a great source for projects / ideas 😍 pic.twitter.com/MYQgJEPJV2
