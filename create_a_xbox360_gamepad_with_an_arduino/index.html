Create a XBOX360 gamepad with an arduino

I thought it wasn't possible to emulate a XBOX360 gamepad, turns out it is possible!

https://www.partsnotincluded.com/programming/arduino-xinput-library/ https://github.com/dmadison/ArduinoXInput

Most of the time, Arduino library emulate DirectInput Gamepad, but most games won't work with it, as Microsoft forced the XINPUT standard to sell more gamepad on PC.
